Prerequisites:
   DB : My Sql installation

   Python :
	pip install fastapi
	pip install uvicorn
	pip install pymysql
	pip install SQLAlchemy
    pip install requests

run : under RubikCrudDemo main folder
	python main.py
	    application runs on localhost:8000 address

test:
	python manuel_test.py : for running from terminal
	test directory : for running for IDE

Note : Project is created via JetBrains PyChram tool and 
web server libraries automatically added which are not added here.
"venv" library directory is also should be added to run the project
